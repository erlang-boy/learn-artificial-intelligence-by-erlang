##二、人工智能历史
----------------------------

### 人工智能分类：
*  强人工智能：关注能够模拟人类思维的人工智能，关注与自顶向下的求解方法，试图模拟人脑的高层次概念（规划，推理及语言理解）
*  弱人工智能：关注特定的目标，求解特定的问题，自底向上的方法，在低层次概念上进行学习
